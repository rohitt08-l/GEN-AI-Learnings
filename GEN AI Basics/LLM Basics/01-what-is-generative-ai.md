# What is Generative AI?

## Introduction

Generative AI is a type of Artificial Intelligence that can generate new content such as:

- Text
- Images
- Audio
- Video
- Code

Unlike traditional AI systems that mainly classify or predict data, Generative AI creates entirely new outputs based on patterns learned from massive datasets.

---

# Traditional AI vs Generative AI

| Traditional AI | Generative AI |
|---|---|
| Predicts outputs | Creates new content |
| Classification | Generation |
| Detect spam emails | Write emails |
| Predict house price | Generate house designs |

# How Generative AI Works

Generative AI models learn patterns from huge datasets.

Example:
- Books
- Websites
- Articles
- Images
- Code repositories

The model learns:
- language structure
- grammar
- context
- relationships between words

Then it generates human-like responses.

# Advantages of Generative AI

- Automates repetitive tasks
- Saves time

---

# Limitations of Generative AI

- Can generate incorrect information
- Hallucinations
- Bias in outputs
- Security risks
- Requires large computational power

# Summary

Generative AI is one of the most important technologies in modern AI. It enables machines to generate human-like content including text, images, audio, video, and code.

It is widely used in chatbots, coding assistants, content creation, healthcare, and automation systems.
